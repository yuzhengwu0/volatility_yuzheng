# Project Notes

This folder contains the current code, figures, and older exploratory scripts for the volatility-confidence analysis project.

## Folder structure

### `core_code/`
All of the figures currently used in the project were generated from the scripts in `core_code/`.  
This folder contains the main analysis code that reflects the current workflow.

### `figure/`
This folder contains many exported figures and different versions of those figures.  
Because I did not label or organize figure versions very carefully at the time, it is not always easy to fully track which file corresponds to which intermediate stage. Some filenames may also be truncated or look inconsistent because the automatically generated PDF outputs were not always easy to format cleanly.

That said, the figures currently placed in my Overleaf document are the **latest versions**, including the current **big graphs** as well as the other final figures used in the write-up.

### `old/`
This folder contains older step-by-step scripts, exploratory analyses, and trial code from earlier stages of the project.  
These files are kept mainly for reference, even though they are no longer part of the current workflow and are generally not in use anymore.

## Additional note
Some PDF figures in this repository may appear slightly cut off. This is mainly due to limitations in the automatic figure export / PDF generation process rather than intentional formatting.
